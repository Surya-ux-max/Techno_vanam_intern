Colors

v1.0

The Athlixir color system is built for high contrast dark UI, performance & energy, sports-tech identity, dashboard readability, and accessibility compliance. Visual tone: Aggressive, Modern, AI-driven, Athletic, Premium dark SaaS.

Color Usage Rules
  • Primary = Action only (use < 10% of total UI area)
  • Backgrounds must use surface layering (never pure black #000000)
  • No random hex values - use tokens only
  • No gradients in dashboard UI
  • No neon effects or colored shadows
  • All text must meet 4.5:1 contrast minimum

Illustration Style

v1.0

Athlixir illustrations must be modern, minimal, data-driven, youth-focused, and professional. Use flat vector style with soft rounded edges and clean geometric forms.

Style Direction

✓ Use
  • Flat vector style
  • Soft rounded edges
  • Subtle depth (light shadow)
  • Clean geometric forms
  • Minimal detail
✗ Avoid
  • Heavy gradients
  • 3D realism
  • Excessive detail
  • Overly playful cartoons
  • Mixed stroke thickness
Line Treatment

Consistent stroke weight • Rounded stroke ends • Smooth curves • No mixed thickness

Color Usage

Color Hierarchy
60% Neutral
30% Brand primary
10% Accent color

Restrictions: Max 4 colors per illustration • No neon colors • No clashing contrast • Use semantic colors consistently

Background Rules

✓ Allowed
  • Transparent
  • Light neutral background
  • Subtle gradient (marketing only)
✗ Not Allowed
  • Busy patterns
  • Heavy textured backgrounds
  • Photographic overlays
  • Hard shadows

Component States

v1.0

All interactive components must support defined states for consistent user feedback and accessibility compliance.

Default State

Default Chip

Standard appearance • No elevation change • Neutral border

Hover State

Background slight change • Shadow increase (if applicable) • No size change

Active / Pressed State

Active Tab

Slight pressed effect • Darker background • No layout shift

Focus State

2px focus ring • Accessible contrast • No shadow glow

Disabled State

Disabled Chip

40–60% opacity • No hover • No shadow • Cursor: not-allowed

Loading State

Processing...

Spinner centered • Button text hidden or replaced • Disabled interaction • No layout shift

Error State

Please enter a valid email address

Red border (2px) • Error text (12px) • Optional icon • No shaking animation

Success State

Success
Your changes have been saved successfully

Green highlight • Success icon • Subtle background tint • Clear confirmation text
